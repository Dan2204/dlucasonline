Website Driving School

Driving instructor website

The Project...

Matt approached me about building his website for him. A recovering alcoholic, he found solace in writing poetry. This site is a poetry site, complete with an administration dashboard, allowing Matt to upload poems and images wherever he is. People can register, sign in and leave comments on poems. The admin dashboard can manage all of this functionality.

This webiste was build with Flask on the backend using SQLAlchemy for the database management. Front end utilized Bootstrap for the navigation bar and grid layout.

Languages: Python Flask, HTML, Css, Javascript
